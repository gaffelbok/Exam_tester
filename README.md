# Modern Exam Tester

A Next.js application designed to help you prepare for certification exams. It features a modern, minimalistic UI and uses Google Gemini AI to automatically generate exams from Word (.docx) or PDF documents.

## Features

- **Modern UI:** Clean, minimalistic interface built with Tailwind CSS.
- **Local Database:** All exams and questions are stored locally in an SQLite database.
- **AI-Powered Migration:** Automatically migrates legacy exam data from JavaScript files.
- **AI Document Scraping:** Upload your study materials (PDF/Word) and let Gemini @2.5-flash extract questions, options, and answers.
- **Smart Explanations:** Missing explanations are automatically generated by AI (marked with a disclaimer).

## Getting Started

### 1. Install Dependencies
```bash
npm install
```

### 2. Configure API Key
Create a `.env.local` file in the root directory and add your Gemini API Key:
```env
GEMINI_API_KEY=your_actual_api_key_here
```

### 3. Migrate Legacy Data (Optional)
If you have existing exams in the `legacy/` folder, run the migration script:
```bash
node migrate-legacy.mjs
```

### 4. Run the App
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

- **Frontend:** Next.js 15 (App Router), React, Tailwind CSS, Lucide Icons.
- **Backend:** Next.js API Routes, SQLite (better-sqlite3).
- **AI:** Google Generative AI (Gemini 2.5-flash).
